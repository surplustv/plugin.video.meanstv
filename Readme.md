## Development

### Requirements

* [pipenv](https://pypi.org/project/pipenv/) (install: `pip install pipenv`)

### Setup

```bash
pipenv install -d
```

This will create a virtual environment and install the required dependencies.

### Testing

Linting the code

```bash
pipenv run lint
```

Running the unit tests

```bash
pipenv run test
```
